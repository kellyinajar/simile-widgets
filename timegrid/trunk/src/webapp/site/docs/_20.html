Date asString
Returns a string representation of the date object according to Date.format. (Date.toString may be used in other places so I purposefully didn't overwrite it)

Date fromString
Returns a new date object created from the passed String according to Date.format or false if the attempt to do this results in an invalid date object (We can't simple use Date.parse as it's not aware of locale and I chose not to overwrite it incase it's functionality is being relied on elsewhere)
var dtm = Date.fromString("12/01/2008");
dtm.toString();
